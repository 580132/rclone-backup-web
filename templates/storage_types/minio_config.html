MinIO特性：
  • 高性能：专为高吞吐量和低延迟设计
  • S3兼容：完全兼容Amazon S3 API
  • 云原生：支持Kubernetes和容器化部署
  • 开源：Apache License 2.0开源协议
配置说明：
  • 端点地址通常包含端口号（默认9000）
  • 确保MinIO服务器可以从备份系统访问
  • 建议创建专用的Access Key用于备份
  • 存储桶需要预先创建或确保有创建权限
