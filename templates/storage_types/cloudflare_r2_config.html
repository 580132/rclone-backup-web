Cloudflare R2配置 - 使用S3兼容协议访问Cloudflare R2存储
R2 API Token的Access Key ID
R2 API Token的Secret Access Key
R2端点地址，不包含https://前缀
注意：
  • Cloudflare R2的区域设置为auto
  • 端点格式为：account-id.r2.cloudflarestorage.com
  • 需要在Cloudflare控制台创建R2 API Token
获取API Token：
  1. 登录Cloudflare控制台
  2. 进入R2对象存储
  3. 点击"管理R2 API令牌"
  4. 创建新的API令牌并获取Access Key和Secret Key
